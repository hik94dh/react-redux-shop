# Интернет-магазин на React/Redux

##  [Демо](https://shop-react-htdrlscwqk.now.sh)

## Установка

```
$ yarn
$ yarn start
```

## В данном проекте работает:

* загрузка данных с API;
* подгрузка товаров;
* поиск;
* фильтр;
* корзина(пустая и с товарами);
* у каждого товара есть своя карточка с информацией;
